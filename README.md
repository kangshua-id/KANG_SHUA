KANG_SHUA
KANG_SHUA adalah galeri digital dan showcase desain karakter yang terinspirasi dari budaya Kupang dan aksara Bali. Proyek ini menampilkan ilustrasi, storytelling visual, dan animasi web interaktif yang dibangun dengan HTML, CSS, dan JavaScript.

Struktur Proyek
html/ — halaman utama seperti index.html, profile.html, showcase.html

html2/ — halaman tambahan seperti art.html, game.html, detail-UI.html

css/ — styling modular untuk tiap halaman

js/ — animasi dan interaksi berbasis JavaScript

aset/image/ — ilustrasi dan aset visual

Fitur Utama
Scroll animation dan cinematic text

Layout responsif dan modular

Showcase carousel dan UI detail

Branding visual khas KANG_SHUA
