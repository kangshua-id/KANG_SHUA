# KANG_SHUA
Proyek ini berisi galeri ilustrasi dan desain karakter oleh KANG_SHUA, terinspirasi dari budaya Kupang dan aksara Bali. Dibangun dengan HTML, CSS, dan JavaScript, menampilkan animasi scroll, layout responsif, dan storytelling visual yang membumi. Cocok untuk showcase dan eksplorasi UI kreatif.
